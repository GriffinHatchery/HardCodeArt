Hard Code Art

This is a project that is aimed at using Python and Neural Networks to understand art.

Download as .zip Download as .tar.gz View on GitHub

Background:

Sound is able to convey emotions that are more then just cultural. For music it seems most reasonable that this has everything to do with the frequencies that make it up. Using the Fourier Transform it is possible to more any functioninto the frequencies domain. This includes images.

Theory:

Art is able to convey certain emotions and thoughts without evoking cultural memes. This has something to do with the frequencies and beat frequencies that make up the art.

Model:

To test my theory I created a neural network which I will train it to recognize adjectives people associate with art. The images will be transformed to the frequencies domain using PyBrain's Fast Fourier Transform. This will be used as the inputs to the Neural Network. The outputs will be a set of adjectives. I narrowed down the emotions to: Sad, Happy, Serious, Whimsical, Peaceful, Intense, Angry, and Simple. Only images without recognizable objects it will be used to insure that the emotions are caused by the colors and patterns not the objects.

Why Should You care:

If the theory is right then it could be used to better understand how we interact with our environment. Surly Artiest and Advertisers already are taping into this (Intentionally and unintentionally). If we were able to have a formal understanding of how this works, producers of images would be better able to utilize the subconscious connections we make. Consumers could be made more aware of how they are emotionally effected by Art.
